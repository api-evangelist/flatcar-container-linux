# Flatcar Container Linux (flatcar-container-linux)
Flatcar Container Linux is a CNCF incubating minimal, immutable Linux distribution optimized for running containers. It features automatic atomic updates and supports Kubernetes on bare metal, cloud, and virtual environments.

### Flatcar Nebraska Update API
Nebraska provides an API for managing update channels, groups, and packages for Flatcar Container Linux instances using the Omaha protocol.
